# 手势控制电脑桌面应用

这是一个基于Python的手势识别桌面应用，可以通过摄像头识别手势来控制电脑的鼠标、键盘操作。

## 功能特性

- 🖱️ **鼠标控制**：通过食指移动控制鼠标指针
- 👆 **点击操作**：握拳手势进行左键点击
- 📜 **滚动控制**：大拇指向上/向下进行页面滚动
- ⌨️ **键盘控制**：剪刀手控制空格键，张开手掌控制ESC键
- 🎥 **实时预览**：实时显示摄像头画面和手势识别结果
- 🎛️ **模式切换**：支持鼠标控制和键盘控制两种模式

## 手势说明

### 鼠标控制模式
- **食指移动**：控制鼠标指针位置
- **握拳**：执行左键点击
- **大拇指向上**：向上滚动页面
- **大拇指向下**：向下滚动页面

### 键盘控制模式
- **剪刀手(✌)**：按下空格键（常用于暂停/播放）
- **张开手掌**：按下ESC键（常用于退出）

## 安装要求

### 系统要求
- Windows 10/11
- Python 3.7+
- 摄像头设备

### 依赖包
```bash
pip install -r requirements.txt
```

主要依赖：
- `opencv-python`: 摄像头捕获和图像处理
- `mediapipe`: 手势识别
- `pyautogui`: 鼠标键盘控制
- `tkinter`: 图形用户界面
- `Pillow`: 图像处理

## 使用方法

1. **安装依赖**
   ```bash
   cd gesture_control_app
   pip install -r requirements.txt
   ```

2. **运行应用**
   ```bash
   python main.py
   ```

3. **操作步骤**
   - 点击"启动摄像头"按钮
   - 确保手部在摄像头视野范围内
   - 选择控制模式（鼠标控制/键盘控制）
   - 使用相应手势进行操作

## 注意事项

- 确保摄像头权限已开启
- 保持手部在摄像头视野范围内
- 手势识别需要良好的光线条件
- 建议在距离摄像头30-60cm的位置使用
- 首次使用可能需要调整手势识别的敏感度

## 故障排除

### 摄像头无法启动
- 检查摄像头是否被其他应用占用
- 确认摄像头驱动已正确安装
- 检查Windows隐私设置中的摄像头权限

### 手势识别不准确
- 确保光线充足
- 保持手部与摄像头距离适中
- 避免背景过于复杂
- 手势动作要清晰明确

### 控制响应延迟
- 关闭其他占用CPU的应用
- 检查系统性能
- 调整摄像头分辨率
